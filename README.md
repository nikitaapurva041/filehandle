Text File Editor (Python Tkinter)

A simple and user-friendly Text File Editor GUI application built using Python and Tkinter. This app allows users to open, edit, replace text, and save .txt files easily.

Features

📂 Open text files (.txt)
✏️ Edit file content in a text area
🔍 Find and replace words
💾 Save changes to existing file
📁 Save file as a new file
⚠️ Confirmation before overwriting files
🖥️ Clean and simple GUI

Tech Stack
  Python 3
  Tkinter – GUI development
  Built-in modules:
    filedialog
    messagebox

 Project Structure
  text-file-editor/
  │
  ├── app.py        # Main application
  └── README.md     # Documentation   

Installation
Clone the repository
  git clone https://github.com/YOUR-USERNAME/text-file-editor.git
  cd text-file-editor
Run the application
  python app.py

How to Use
Click Open File to load a .txt file
Edit the content in the text area
Use Find and Replace fields to modify text
Click:
  Replace → Replace all matching words
  Save File → Save changes to current file
  Save as File → Save as a new file
